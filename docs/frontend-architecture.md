# Integration Guide

CSAC-Lite Verifier is designed for simple adoption in review, audit, and reporting workflows.

## User flow

1. Submit a verification package.
2. Review the visible package information.
3. Receive a clear status.
4. Present the outcome in a report or dashboard.

## Interface role

The interface helps users:

- load a package;
- review metadata;
- launch a check;
- read a status;
- prepare a summary.

## Reviewer experience

The result should be easy to understand, easy to share, and easy to include in an operational process.

## Principle

The public experience should focus on value, clarity, and trust.
