# Status Codes

CSAC-Lite Verifier presents clear statuses for reviewed packages.

## VERIFIED

The submitted package passed the configured review checks.

## INVALID

The submitted package did not pass the expected review checks.

## UNSUPPORTED PROFILE

The submitted package references a profile that is not supported in the current evaluation context.

## REVIEW REQUIRED

The package may require additional human or organizational review.

## Principle

Statuses should be short, readable, and easy to include in reporting workflows.
