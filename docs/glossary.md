# Glossary

## Verification package

A structured file submitted for review through CSAC-Lite Verifier.

## Verification status

A short result that summarizes the outcome of a review.

## Reviewer

A person or team responsible for assessing the status of an important digital result.

## Operator

An organization or platform using CSAC-Lite Verifier in a decision workflow.

## Audit-ready summary

A readable summary that can support reporting, governance, or compliance review.

## Evaluation

A controlled review phase before production use.
