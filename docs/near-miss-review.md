# Near Miss Review

A near miss is a warning signal that did not become a visible failure.

Genesis Verified treats near misses as useful review material.

## Examples

- unclear status interpretation;
- missing context;
- unusual reviewer hesitation;
- repeated escalation;
- mismatch between outcome and operating situation.

## Purpose

Near miss review helps organizations learn before a decision failure becomes visible.

## Governance value

Weak signals should not disappear only because no final failure occurred.

## Principle

A responsible review system should learn from signals before they become incidents.
