# Relevance Layer

The relevance layer helps reviewers focus on what matters.

A complete outcome is not always a relevant outcome.

## Purpose

The layer helps teams distinguish between available information and useful review material.

## Review questions

- Does this outcome matter for the decision at hand?
- Does the context still support the status?
- What should the reviewer see first?
- What can remain secondary?

## Principle

Review should begin with relevance, not volume.
