# Immediate Readout

Immediate readout gives reviewers a first orientation without turning speed into blind trust.

## Purpose

The readout helps teams see the current status, context and review need quickly.

## What it should provide

- current status;
- relevant context;
- visible uncertainty;
- challenge route;
- next review step.

## Boundary

Immediate does not mean final.

It means the review starts without delay.

## Principle

Instant visibility should support review, not replace judgement.
