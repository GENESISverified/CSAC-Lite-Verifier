# Roadmap

This public roadmap describes the direction of CSAC-Lite Verifier without committing to delivery dates.

## Current focus

- Public service presentation
- Demonstration examples
- Documentation for adoption
- Operator evaluation workflows

## Next focus

- Clearer reviewer guidance
- Stronger reporting templates
- Improved status explanations
- Better onboarding material

## Future direction

- Managed evaluation support
- Advanced reporting workflows
- Sector-specific documentation
- Institutional adoption resources

## Principle

The roadmap prioritizes clarity, trust, and responsible adoption.
