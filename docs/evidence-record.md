# Evidence Record

An evidence record helps reviewers understand why a status was assigned.

It does not need to expose unnecessary internal detail.

## What it can include

- outcome reference;
- review status;
- context note;
- reviewer note;
- escalation note;
- reporting summary.

## Purpose

The record helps teams explain what was reviewed and what remained uncertain.

## Review value

Evidence should support responsible review, not create false certainty.

## Principle

A status is stronger when the basis for review can be understood later.
