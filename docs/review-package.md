# Review Package

A review package is the object submitted for structured review.

It should be understandable, limited and useful for evaluation.

## Purpose

The package helps reviewers see what outcome is being reviewed and what status can be assigned.

## Public evaluation fields

A public evaluation package may include:

- outcome identifier;
- review profile;
- status information;
- references;
- metadata;
- summary.

## Review value

The package is not the decision itself.

It is the material used to support review, challenge, escalation and recording.

## Principle

A critical outcome should be packaged in a way that makes review possible without exposing unnecessary internal detail.
